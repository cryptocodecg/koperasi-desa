## 🎮 Tentang Game

**KopDesa** adalah game simulasi berbasis web yang memandu pemain menjadi Ketua Koperasi Desa dari nol. Pemain menghadapi keputusan nyata yang dihadapi pengurus koperasi sehari-hari — mulai dari pendirian, pengumpulan modal, penyaluran kredit, hingga pembagian SHU di akhir tahun.

Game ini dirancang sebagai **media edukasi interaktif** yang membuat konsep koperasi mudah dipahami oleh masyarakat umum, terutama generasi muda.

## 🎯 Tujuan

| Masalah | Solusi |
|---|---|
| Literasi koperasi masyarakat masih rendah | Game edukatif yang mengajarkan cara kerja koperasi secara interaktif |
| Koperasi dianggap kuno dan tidak relevan | Pendekatan gamifikasi yang modern dan mobile-friendly |
| Konsep SHU, simpanan, dan kredit sulit dipahami | Simulasi nyata dengan konsekuensi keputusan yang terasa langsung |
| Minat generasi muda terhadap koperasi rendah | Visual colorful, gameplay kasual, bisa dimainkan di browser tanpa install |

---

## 🕹️ Cara Bermain

1. **Pendirian** — Beri nama koperasi, tentukan besaran simpanan, rekrut anggota pendiri, dan gelar rapat pembentukan
2. **Modal Awal** (Bulan 1–3) — Tagih simpanan wajib, alokasikan dana ke instrumen yang tepat (simpan pinjam, unit usaha, deposito)
3. **Operasional** (Bulan 4–9) — Hadapi 4 kejadian nyata: permohonan pinjaman, kredit macet, bencana, digitalisasi, audit Kemenkop
4. **Pengembangan** (Bulan 10–11) — Keputusan strategis: akses dana LPDB, merger koperasi sekunder
5. **RAT & SHU** (Bulan 12) — Hitung surplus, alokasi sesuai UU No.25/1992, dan bagikan SHU ke anggota

---

## 📚 Konten Edukatif

Setiap keputusan dilengkapi dengan penjelasan konsep koperasi yang relevan, antara lain:

- Perbedaan simpanan pokok, wajib, dan sukarela
- Cara menghitung dan mendistribusikan SHU
- Prinsip kehati-hatian (prudential) dalam penyaluran kredit
- Klasifikasi NPL (Non-Performing Loan)
- Mekanisme linkage program LPDB-KUMKM
- Struktur organisasi koperasi (Rapat Anggota → Pengurus → Pengawas)
- Kewajiban dana cadangan, pendidikan, dan sosial (UU No.25/1992)
- Manfaat digitalisasi melalui SIKOP dari Kemenkop RI

## 🛠️ Teknologi

| Komponen | Teknologi |
|---|---|
| Frontend | HTML5, CSS3, Vanilla JavaScript |
| Visual | SVG inline (peta desa generatif) |
| Font | Google Fonts — Nunito |
| Hosting | GitHub Pages |
| Dependencies | Tidak ada (zero dependency) |

Game ini berjalan **100% di browser** tanpa framework, tanpa build tools, dan tanpa backend — satu file HTML yang bisa dibuka langsung.

## 🏛️ Kesesuaian dengan Tema Hackathon

Game ini relevan langsung dengan tema **digitalisasi koperasi** karena:

1. **Edukasi digital** — mengajarkan cara kerja koperasi kepada masyarakat melalui media interaktif
2. **Meningkatkan literasi koperasi** — konten mengacu pada UU No.25/1992 dan panduan Kemenkop RI
3. **Mempromosikan SIKOP** — salah satu keputusan dalam game mendorong pemain mengadopsi sistem digital resmi Kemenkop
4. **Menarik minat generasi muda** — format game kasual mobile-friendly yang mudah diakses siapa saja
5. **Skalabel** — dapat dikembangkan menjadi platform edukasi koperasi nasional
